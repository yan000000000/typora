# math

## setting

### math block

*   click "paragraph" --> "math block"
*   Ctrl + Shift + m
*   "$$" + enter

### inline math

*   file --> preference --> markdown --> inline Math

>   tips: press \$ and Esc can automatically add a \$ behind
